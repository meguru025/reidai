文字タグの見本帳（HTML Typo Cheatsheet）

このページは、見出し・本文・強調・引用・コード・リスト等の代表的な文字系タグをまとめて確認できます。

見出しタグ h1〜h6

H1 見出し（ページの主見出し）

H2 見出し（章）

H3 見出し（節）

H4 見出し

H5 見出し
H6 見出し（最小）

段落（p）と改行（br）

これは段落（p）です。文章は段落ごとにまとまります。
ここだけは段落の中で強制改行（br）を使っています。

強調（b / strong / i / em / mark / small）

b は見た目だけ太字、strong は“意味的に重要”な強調。
i は見た目だけ斜体、em は“意味的に”強調（読み上げにも反映）。
mark は蛍光ペン風のハイライト、small は小さな注釈文字。

数式や化学式：x2 + y2、H2O（sup=上付き / sub=下付き）

引用（blockquote / q）

これは blockquote（段落引用）です。文全体を引用として示すときに使います。

— 例：誰かの発言や書籍などの出典

文中の短い引用は このように q タグで示します。

コード（code / pre）

行内コードは <code>console.log("Hello")</code> のように表示。

/* pre は改行や空白をそのまま表示 */
h1 {
  color: cornflowerblue;
  font-size: 40px;
}
